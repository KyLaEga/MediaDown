# Universal Media Downloader

Лёгкое десктоп-приложение для загрузки видео, аудио и фотогалерей практически с любого сайта.

## ✨ Возможности

- 🌐 **Универсальная загрузка** — на базе `yt-dlp` и `gallery-dl`
- 📦 **Поддержка множества форматов** — от MP4 и MKV до MP3 и JPG
- 🧠 **Удобный интерфейс** — современный GUI на PySide6 (тёмная/светлая тема)
- ⏯️ **Управление очередью** — параллельные загрузки с настраиваемой задержкой

## 🚀 Использование (для разработчиков)

```bash
git clone https://github.com/KyLaEga/MediaDown.git
cd MediaDown

pip install -r requirements.txt
python main.py
```
